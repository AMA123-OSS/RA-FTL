# RA-FTL
本项目针对客户端资源异构、数据非独立同分布的联邦学习场景，提出资源感知联邦迁移学习框架 RA-FTL。在预训练模型稀缺条件下，该框架采用同步预训练择优策略，借助代理验证集筛选最优特征提取层完成迁移与联邦联合训练。依托 MSTAR 数据集实验得到四项结论：1. 偏独立同分布场景中，FTL 性能显著优于 FL，但性能提升伴随随机波动；2. 强非独立同分布条件下，预训练客户端数据偏离全局分布会造成特征提取器偏置，冻结过多网络层不利于客户端适配本地数据；3. 资源高度异构时，预训练客户端数量存在最优下限区间，并非数量越大性能越好，实验中 2 个预训练客户端效果最优；4. 资源感知竞争预训练能生成高表征性能的特征提取层，加速联邦主模型收敛、提升精度，并有效降低通信开销。
FTL_study_breeze2.0.py为小样本竞争下的资源感知FTL并附有论文手稿pdf文件论文已发表,modiified_async_fed.py为最新研究的异步联邦(论文在撰写)
# 语义分割
语义分割是自动驾驶与智能场景理解的核心任务。现有轻量化实时语义分割模型存在全局语义建模能力不足、精细边界细节丢失严重两大缺陷。为解决上述问题，本文融合频域信息与选择性状态空间模型，提出一种轻量化实时语义分割网络。该网络通过离散小波变换拆分图像低频结构与高频边缘信息，设计小波频域增强模块，构建层级 Mamba 编码器，并结合 FPN 多尺度融合策略优化特征提取。公开数据集实验结果表明，该方法能够有效平衡分割精度、模型复杂度与实时推理速度，综合性能更优。
